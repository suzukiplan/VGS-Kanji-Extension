# ライセンス
GSLOT255.CHRは、門真 なむ様の8×12 ドット日本語フォント「k8x12」のk8x12S（2015年1月31日時点のもの）に含まれる7bit ASCII（JIS X 0201の内0x20～0x7Fの範囲）のフォントデータをvgsbmpコマンドで変換したものです。
